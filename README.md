### Installation
Installing dependencies 
```sh
> cd client
> npm install
```
```sh
> cd ..
```
```sh
> cd server
> npm install
```

### Running the application
Client | http://localhost:3000/
```sh
> npm start
```

Server | http://localhost:3001/
```sh
> node index.js
```

Nodemon 
```sh
> npm run devStart
```
